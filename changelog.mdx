<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors in documentation themes. Set your `{brandColor}` in the config to match `#3B82F6`.
- Introduced `<Video>` component for embedding self-hosted and platform videos directly in MDX pages.

## Improvements

- Enhanced Mermaid diagram rendering with better error handling for complex graphs.
- Optimized page load times by `>30%` through lazy-loading of JSX components.

## Bug Fixes

- Fixed parsing errors when using nested code blocks inside `<Tabs>`.
- Resolved issue with `<ParamField>` displaying incorrect `param-type` for query parameters.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- New `<Columns>` component supports up to 5 columns for flexible layouts.
- Added `tags` prop to `<Update>` components for better categorization in changelogs.

## Bug Fixes

- Corrected escaping of `{` and `}` characters in prose text to prevent MDX parsing failures.
- Fixed `<Steps>` icon rendering on mobile devices.

## Documentation Updates

- Expanded component reference with syntax checklists and verification rules.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Initial release of core MDX components: `<Callout>`, `<Card>`, `<CodeGroup>`, and more.
- Full support for YAML frontmatter parsing without blank line errors.

## Breaking Changes

- Removed legacy `<Mermaid>` JSX component. Use fenced code blocks with ````mermaid```` syntax only.

## Improvements

- Added 20+ Lucide icons for all components.

</Update>
